# FlowSFTP

**FlowSFTP** is a **WinSCP-inspired**, **cross-platform** desktop SFTP client with a **simple, uncluttered UI**. It brings the familiar two-pane commander workflow and session-style login to **macOS, Windows, and Linux**—without the visual noise of heavier tools.

**Milestone M1** focuses on SFTP: connect, browse, transfer files, and manage sessions in a lightweight desktop app.

## Features

- **Cross-platform** — runs on **macOS**, **Windows**, and **Linux**
- **Commander layout** — **WinSCP-style** dual panes (local + remote), paths, refresh, and parent folder navigation
- **SFTP** — connect with password and/or **private key**; **Test** a session before logging in
- **Site manager** — save named sites, optional groups, search saved sites, **Import** / **Export** session backups
- **File transfers** — multi-select **upload** and **download**, **drag-and-drop** between panes, context menu actions, **F5**-style transfer of the focused selection, and a **transfer queue**
- **Multi-window** — **File → New window** or **Ctrl/Cmd+N** for a separate commander; each window keeps its own connection until you close it
