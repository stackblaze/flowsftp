# FlowSFTP keyboard shortcuts

Milestone **M1** implements navigation via the UI (toolbar buttons). WinSCP-style shortcuts (F5 copy, F6 move, etc.) are planned for milestone **M7**.

| Action              | M1                         |
| ------------------- | -------------------------- |
| New window          | **File → New window**, **Ctrl+N** / **⌘N** (OS menu) |
| Close window        | **File → Close window** or **Ctrl+W** / **⌘W** (OS) |
| Refresh local pane  | Local **↻** button         |
| Refresh remote pane | Remote **↻** button        |
| Parent directory    | **↑** on each pane toolbar |
| Apply path in field | Enter or blur path input   |
